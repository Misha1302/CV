Projects

UniversalToolchain / Wist

GitHub Docs NuGet

.NET 10 · compiler-first DSL/runtime framework

A framework for building embeddable domain-specific languages. Wist is the reference language used to validate the generic architecture and semantic equivalence across execution modes.

  • Source → Lexer/Parser → AST → Bytecode → AIR → optimizers → backend → execution.
  • AIR interpreter and compiled typed execution through CIL/DynamicMethod.
  • Composable language modules and declarative dialect definitions.
  • Backend capabilities, intrinsics, IR optimizations and cross-backend parity regressions.
  • CLI, application facade, NuGet preview, GitHub Actions, package smoke tests and VitePress documentation.
  • BenchmarkDotNet and generated CIL/assembly analysis for hot-path optimization.
