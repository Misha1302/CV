Compiler/runtime framework

UniversalToolchain / Wist2

Public modular .NET platform for defining DSLs and executing programs: dialect definitions, lexer/parser/AST, bytecode/AIR, optimizers, an AIR interpreter and typed CIL generation through DynamicMethod. The platform lets a .NET application accept restricted formulas and DSLs, validate them before execution, interpret them for diagnostics, and compile supported hot paths into typed delegates.

GitHub ↗ Documentation ↗
  • Architecture: manifest-backed runtime selection, deterministic module composition and capability contracts instead of a hardcoded backend
  • Correctness: semantic parity across interpreter/CIL, negative cases, release gates and documented limitations
  • Research direction: callable-first SSA as an optional IR capability; BenchmarkDotNet and hot-path analysis without overclaiming
Program analysis

PS-form Memory Dependence Analyzer

Standalone C17 solver for classifying overlaps between parametric memory accesses across independent loop iterations.

  • Ranked 1st among 49 submissions; 5.0/5.0 and 104/104 official tests.
  • Conservative yes/no/maybe semantics that return “maybe” when a proof is unavailable.
  • Symbolic normalization, range/residue/GCD, exact affine analysis and bounded exact search.
