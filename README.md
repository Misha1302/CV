## Проекты

### [UniversalToolchain / Wist2](https://github.com/Misha1302/Wist2)

Модульный .NET-фреймворк для создания DSL и runtime-пайплайнов. Wist — эталонный язык внутри проекта, на котором проверяется архитектура.

Что есть в проекте:

- пайплайн: исходный текст → лексер → парсер → AST → bytecode → IR → backend;
- два backend-направления: interpreter и CIL/DynamicMethod compiled execution;
- IR-оптимизаторы и backend-aware intrinsics;
- тесты на совпадение семантики interpreter и compiled execution;
- BenchmarkDotNet-бенчмарки;
- профилирование и анализ узких мест;
- developer-документация через VitePress.
