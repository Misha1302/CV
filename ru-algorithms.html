Графовые алгоритмы · C++23

MCST Tasks on Graph Algorithms

Общая модель ориентированного графа и четыре самостоятельных алгоритмических программы: RPO с поиском циклов, Dijkstra, Dinic maximum flow и Tarjan SCC. Репозиторий использует строгие предупреждения, санитайзеры, статический анализ и автоматическую проверку тестов.

GitHub ↗
  • Сложность: линейные обходы для RPO/SCC, очередь с приоритетом для кратчайших путей, остаточная сеть и blocking flow для Dinic.
  • Инженерия: общее ядро графа, CMake, C++23, ASan/UBSan, clang-tidy, тесты .in/.out.
  • Краевые случаи: циклы, недостижимые вершины, self-loops, параллельные рёбра и несколько компонент.
