Проекты

UniversalToolchain / Wist2

Модульный .NET-фреймворк для создания DSL и runtime-пайплайнов. Wist — эталонный язык внутри проекта, на котором проверяется архитектура.

  • Пайплайн: исходный текст → лексер → парсер → AST → bytecode → IR → backend.
  • Два backend-направления: interpreter и CIL/DynamicMethod compiled execution.
  • IR-оптимизаторы, backend-aware intrinsics, тесты на совпадение семантики.
  • BenchmarkDotNet-бенчмарки, профилирование, анализ узких мест и developer-документация через VitePress.

Unity/C# проекты

Unity/C# — мой сильный практический стек до перехода к более системным .NET-проектам. Делал законченные игровые и образовательные приложения: проекты для курсовых работ, party games для детских праздников, викторины и тренажёр таблицы умножения.
