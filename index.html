О себе

Уже несколько лет изучаю архитектуру компиляторов, языков программирования и runtime-систем. На практике работаю с полным компиляторным конвейером: lexer/parser, AST, bytecode, Abstract IR, оптимизации и два пути исполнения — интерпретатор и CIL-компиляция через DynamicMethod.

UniversalToolchain / Wist2

Документация GitHub

Расширяемый .NET-фреймворк для построения небольших предметно-ориентированных языков. Wist используется как эталонный язык и интеграционная поверхность проекта.

  • Конвейер: source → lexer → parser → AST → bytecode → Abstract IR → optimizers → backend.
  • Интерпретатор Abstract IR и compiled execution через CIL / DynamicMethod.
  • Композиция языковых возможностей через модули и декларативные dialect-конфигурации.
  • Backend-aware intrinsics, IR-оптимизации и тесты semantic parity между способами исполнения.
  • CLI, BenchmarkDotNet-бенчмарки, профилирование и подробная developer-документация.
