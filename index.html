Профиль

Моя основная специализация — compiler/runtime engineering на .NET: языковые frontend-ы, промежуточные представления, интерпретаторы, CIL-кодогенерация, JIT-поведение и оптимизация hot path. При этом я не ограничиваюсь компиляторами: призёрство по промышленному и олимпиадному программированию подтверждает сочетание software engineering и алгоритмической подготовки.

Сильнее всего мне интересны теория графов, теория чисел, low-level и performance-задачи, архитектура сложных систем и R&D, где требуется не только написать код, но и доказать корректность решения.

Проекты

UniversalToolchain / Wist

GitHub Документация NuGet

.NET 10 · compiler-first DSL/runtime framework

Фреймворк для создания встраиваемых предметно-ориентированных языков. Wist — reference language, на котором проверяются универсальность архитектуры и одинаковая семантика разных способов исполнения.

  • Pipeline: Source → Lexer/Parser → AST → Bytecode → AIR → optimizers → backend → execution.
  • Interpreter backend и compiled typed execution через CIL/DynamicMethod.
  • Композиция языковых возможностей из модулей и декларативные dialect-конфигурации.
  • Backend capabilities, intrinsics, IR-оптимизации и semantic-parity regression tests.
  • CLI, public facade API, NuGet preview, GitHub Actions, package smoke tests и VitePress-документация.
  • BenchmarkDotNet и анализ generated CIL/assembly для оптимизации горячих путей.
