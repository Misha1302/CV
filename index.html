Технические работы

UniversalToolchain / Wist

GitHub Документация NuGet

.NET 10 · modular DSL/compiler/runtime framework

Framework для построения встраиваемых DSL. Wist служит reference language и proving ground для проверки универсальности архитектуры, оптимизаций и одинаковой семантики разных способов исполнения.

  • Спроектировал pipeline Source → Lexer/Parser → AST → Bytecode → AIR → Optimization → Backend → Execution.
  • Разделил frontend-oriented Bytecode и backend-oriented AIR, чтобы языковые модули не зависели от конкретного backend.
  • Реализовал interpreter backend и compiled typed execution через CIL/DynamicMethod.
  • Перевёл legality оптимизаций с hardcode конкретных backend-ов на capabilities и supported intrinsics.
  • Защитил семантику cross-backend parity, module-combination и architecture-guardrail тестами.
  • Подготовил public facade API, CLI, NuGet preview, package smoke tests, GitHub Actions и VitePress-документацию.
  • Исследую hot path через BenchmarkDotNet, generated CIL и assembly.

Solver и test harness для отбора МЦСТ

C · Python · program analysis

Построил portfolio solver для параметрической зависимости двух обращений к памяти на независимых итерациях цикла при неизвестных loop-invariant переменных.

  • Символьная нормализация сумм произведений и канонизация эквивалентных мономов.
  • Range, residue и GCD abstractions для sound-доказательств отсутствия зависимости.
  • Точный affine solver с подсчётом lattice points через floor_sum.
  • Монотонный two-pointer анализ, ограниченный точный fallback и one-sided witness search.
  • Генеративный harness с exact oracles, metamorphic transformations и mutation-guided поиском WA/TL случаев.
